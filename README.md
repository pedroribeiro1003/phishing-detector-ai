# 🔐 Detector de Phishing com IA

Projeto que utiliza Machine Learning para identificar URLs maliciosas.

## 🚀 Tecnologias
- Python
- Flask
- Scikit-learn
- HTML + JavaScript

## 💡 Funcionalidade
O sistema analisa URLs e classifica como:
- Seguro
- Phishing

```bash
pip install -r requirements.txt
python app.py
